# 🏗️ Site Web — Céline Morizet Architecte

## 📌 Description du projet

Ce projet consiste en la conception et le développement d’un site vitrine professionnel pour **Céline Morizet, architecte DPLG**, basé en Bretagne.

L’objectif est de proposer une plateforme moderne permettant de présenter son activité, ses réalisations, ainsi que ses informations légales, tout en offrant une expérience utilisateur fluide et responsive.

---

## 🎯 Objectifs du projet

* Mettre en valeur l’activité d’architecte
* Présenter les projets réalisés (galerie)
* Fournir un point de contact accessible
* Assurer la conformité légale (mentions obligatoires)
* Offrir une navigation claire et intuitive
* Optimiser l’affichage sur tous les supports (responsive)

---

## 🛠️ Technologies utilisées

* **HTML5** — structure sémantique des pages
* **CSS3** — design, mise en page et responsive
* **JavaScript** — interactions dynamiques (galerie, UI)
* **VS Code** — environnement de développement
* **Git & GitHub** — versioning du projet
* **Render** — déploiement du site
